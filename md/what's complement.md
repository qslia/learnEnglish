### What is a complement?

A **complement** is a word or phrase that **gives more information about another word** and makes the meaning complete.

### In your sentence:

> **I like listening to birds singing.**

* **birds** = object
* **singing** = **object complement**

### Simple definition (exam-friendly):

> **An object complement describes or completes the meaning of the object.**

### Common pattern:

* **see / hear / notice + object + doing**

* **make / let / call + object + complement**
